Game Rules & How to Play

  • The game has 5 rounds, each with increasing difficulty: Easy, Medium, Hard, Very Hard, Super Hard.
  • Each round, a set of numbers will be shown for a few seconds. Memorize them!
  • After the countdown, the numbers will be hidden. Enter them in order, separated by commas.
  • You have limited time to input your answer. The harder the level, the less time you get!
  • Use the 💡 Hint button to reveal one number (costs 2 points, only once per round).
  • Try to get the highest score possible. Good luck!
